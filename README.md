# Web Technologies Question Bank

A mini project containing programs of:

- HTML
- CSS
- JavaScript
- PHP

The project contains a collection of 5 questions from each technology with:

- Source Code
- Output Display
- Navigation System

---

- HTML5

- JavaScript

- Vercel

---

# How to Run

## HTML/CSS/JS

Open with Live Server in VS Code.

---

## PHP

1. Install XAMPP
2. Move project into:

```text
C:\xampp\htdocs\
```

3. Start Apache
4. Open:

```text
http://localhost/web-tech-project
```
